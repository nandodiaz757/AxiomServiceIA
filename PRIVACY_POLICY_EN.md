# Privacy Policy for AxiomQA (Accessibility UI Change Analyzer)

**Effective Date:** November 30, 2025

---

## 1. Introduction

AxiomQA is a professional tool designed for QA teams and app owners to analyze and compare changes in the user interface (UI) of Android applications across different versions. This Privacy Policy explains what data we collect, how we use it, and how we protect your information.

---

## 2. What Data We Collect

**AxiomQA ONLY collects:**
- The structure of the app's user interface (UI):
  - Button, text, image, and layout elements
  - Accessibility properties (labels, hints, enabled/disabled states)
  - The order and hierarchy of UI components
  - Non-personal metadata (screen names, version numbers)
- Event types and timestamps (for analysis of UI changes)

**AxiomQA DOES NOT collect:**
- Personal user data (names, emails, phone numbers, addresses)
- Passwords, credit card numbers, or sensitive input fields
- Any content entered by users in forms or text fields
- Any data from outside the app's UI structure

---

## 3. How We Use Your Data

- To compare UI changes between app versions
- To generate automated QA reports and dashboards
- To help testers and app owners detect unintended changes (e.g., missing buttons, added elements)
- To improve the accuracy of our change detection algorithms

**We DO NOT use your data for:**
- Advertising or marketing
- Selling to third parties
- Profiling or tracking users
